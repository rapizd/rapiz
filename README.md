| <sub>IN THIS SECTION</sub> |

| - Install with pip<br />- Create your first app |

You can install Rapiz via [pip](https://pypi.org/project/rapiz), like so:

```haskell
$ pip install rapiz
```

Then create your very first Rapiz Application:

```haskell
$ rapiz new hello
```
